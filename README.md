# Site sobre Super-Heroi

Atividade sugerida em aula com o objetivo de estudar frameworks, como o Bootstrap.
Ela consistia em escolher um herói e criar um site explorando as ferramentas desse framework.
No caso deste projeto, o herói escolhido foi Visão.
